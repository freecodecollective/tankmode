# TANKMODE

A fast, modern browser arena shooter (diep.io / Geometry Wars style) packed into a single self-contained HTML file. No install, no server — runs offline by opening `index.html`, and online via GitHub Pages.

**Play:** https://freecodecollective.github.io/tankmode/

## Controls
- **WASD / Arrows** — move
- **Mouse** — aim
- **Hold Left-Click / Space** — auto-fire
- **1–8** — spend skill points on upgrades
- **P / Esc** — pause · **M** — mute

## What it is
Farm the drifting shapes for XP, level up and pour points into 8 stats, evolve your tank down a branching class tree (Twin, Sniper, Machine Gun, Flank Guard… and on into 25 classes), and climb the live leaderboard past AI bots that hunt, lead their shots, dodge, kite, and retreat. Scrolling arena with a minimap, particle effects, screen shake, and kill feed.

Single file: `index.html` (HTML/CSS/JS inline, zero dependencies).
